  4. Buat design database untuk sistem informasi sekolah.
    Database tersebut memiliki tabel siswa, kelas, dan guru yang ter-relasi. Dari data tersebut harus dapat diketahui, siapa wali kelas dari seorang siswa!
    • Membuat tabel siswa
      CREATE TABLE `siswa` ( `nis` int(5), `nama_siswa` varchar(20), `alamat` text, PRIMARY KEY (`nis`) );
    • Membuat tabel guru
      CREATE TABLE `guru` ( `nip` int(5), `nis` int(5), `kode_kelas` int(5), `nama_pengajar` varchar(20), `alamat` text, PRIMARY KEY (`nis`,`nip`,`kode_kelas`) );
    • Membuat tabel kelas
      CREATE TABLE `kelas` ( `kode_kelas` int(5), `nama_kelas` varchar(20), PRIMARY KEY (`kode_kelas`) );
  5. Buat query untuk merelasikan ketiga table tadi, dengan tujuan melihat wali kelas dari seorang siswa
    6. SELECT siswa.nim, siswa.nama_siswa, guru.nama_pengajar, kelas.nama_kelas
    FROM guru
    INNER JOIN siswa
    ON siswa.nim = guru.nim
    INNER JOIN kelas
    ON kelas.kode_kelas = guru.kode_kelas
    WHERE guru.nip = 1001
